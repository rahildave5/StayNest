# 🏡 StayNest – Airbnb Clone

StayNest is a full-stack web application inspired by Airbnb that allows users to explore, list, and book rental properties. It focuses on delivering a seamless user experience with modern UI and secure backend functionality.

---

## 🚀 Features

* 🔐 User Authentication (Signup/Login)
* 🏠 Create, Edit & Delete Property Listings
* 🔍 Search & Filter Listings
* 📅 Booking System
* 💬 Reviews & Ratings
* 📸 Image Uploads
* 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Tools:** Cloudinary, JWT, Git, GitHub, Vercel
